5 Smart & Connected Systems Projects

Fall 2024

This page groups five Smart and Connected Systems projects completed in Fall 2024, all centered on embedded firmware, networking, sensing, and cyber-physical systems.

The project set included Indoor Robot Navigation, E-Voting with Distributed Systems, FitCat, Smart Cat Collar, and Smart Pill.

Together these projects emphasized RTOS-based firmware, ESP32 development, Node.js backends, sensors, networking, and real-time system behavior.

Indoor Robot Navigation System Development
  • Delivered C firmware for an ESP32 microcontroller with RTOS, control algorithms, feedback-based navigation, and wireless remote functionality
  • Created wireless control for manual operation using WASD keyboard inputs
  • Designed obstacle avoidance using ultrasonic and IR sensors, integrating Optitrack position tracking via a Node.js server over UDP
  • Built a 2D visualization of robot paths and waypoints using Streamlit and TingoDB
    • E-Voting System with Distributed Systems
  • Designed an electronic voting system using NFC-enabled fob devices and IR communication for vote transmission
  • Implemented a Bully algorithm for coordinator election to ensure resilience and continuity
  • Developed a Node.js server for vote logging, tabulation, and web visualization on Raspberry Pi
    • FitCat
  • Designed a distributed system of multiple ESP32 cat trackers to transmit data over WiFi to a central server
  • Developed a Node.js-based web portal on Raspberry Pi with live leaderboards, Canvas.js charts, and status updates sent back to trackers
  • Configured a webcam feed into the web portal and established robust networking with static IPs, port forwarding, and DDNS
    • Smart Cat Collar
  • Designed a wearable device using an accelerometer and ESP32 to classify cat activity states and durations while displaying current activity and elapsed time
  • Programmed real-time serial data transmission for time, temperature, and activity visualization in Canvas.js charts
  • Integrated I2C communication, button-triggered displays, and buzzer functionality
    • Smart Pill
  • Developed an embedded system using ESP32 for a diagnostic pill to read, log, and display sensor data in real time
  • Implemented periodic cycles to measure and report temperature, light levels, battery voltage, and tilt in engineering units over a serial interface
  • Designed real-time functionality using an RTOS, hardware interfaces, a voltage divider, and I/O interrupts
