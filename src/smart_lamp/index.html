Smart Lamp

Lutron Light Engineering Competition, Spring 2024

Project Description

Smart Lamp was developed for the Lutron Light Engineering Competition as a responsive lighting device that could automatically turn on and adjust brightness and color based on user distance and room conditions. The project combined embedded firmware, circuit design, sensor integration, and physical prototyping.

My Responsibilities

  • Programmed firmware in C++ with multithreading, enabling user detection at three meters and ten lighting patterns.
  • Designed the circuit using LCD, TMP36, NeoPixels, and ultrasonic sensors.
  • Rapid prototyped the product and created the enclosure using 3D printing and laser cutting in Onshape.

Skills I Learned

  • Sensor-driven embedded system design.
  • Multithreaded firmware development in C++.
  • Prototyping consumer-facing hardware with both functional and aesthetic considerations.
