Open Source Robotic Manipulation System

Spring 2025

This project developed a low-cost robotic manipulation system that combines real-time object detection, multi-camera perception, and LeRobot-controlled 6-DOF arms.

It integrated YOLOv8, embedded vision streaming, dataset collection, behavior cloning, and mechanical integration to build an end-to-end manipulation platform.

This work was accepted to the National Collegiate Research Conference (NCRC) 2026 and presented as a poster at Harvard University.

My responsibilities
  • Developed a low-cost robotic manipulation system integrating real-time object detection with multi-camera visual input and LeRobot-controlled 6-DOF arms
  • Accepted to National Collegiate Research Conference (NCRC) 2026 and presented a poster at Harvard University
  • Designed and implemented an embedded vision pipeline using the Seeed Studio XIAO ESP32S3 Sense module to stream MJPEG video over Wi-Fi for remote inference
  • Built datasets by aligning robot joint states with image frames and bounding boxes to train manipulation policies
  • Trained and deployed behavior cloning models including Diffusion Policy and Action Chunking Transformer with YOLO-based feature augmentation, achieving 50% one-shot success rates in Isaac Sim
  • Designed 3D-printed mounts and mechanical integration for the ESP32 camera system and UVC cameras
  • Contributed to firmware development in ESP-IDF, system calibration, and real-time control synchronization
