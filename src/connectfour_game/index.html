ConnectFour Game

Fall 2022

Project Description

ConnectFour was one of my earliest programming projects and one of my first experiences building a complete software application. I first created a console-based version and then expanded it into an interactive GUI using Pygame.

My Responsibilities

  • Programmed an initial Connect Four prototype in Python with terminal-based output.
  • Developed the final version in Python using Pygame for interactive windowed gameplay.

Skills I Learned

  • Game logic and turn-based state management.
  • Python fundamentals and GUI development with Pygame.
  • Iterating from a simple prototype to a more polished application.
