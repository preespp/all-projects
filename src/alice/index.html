ALICE

AI Life Improvement & Care Expert

MakeMIT x Harvard 2026 Project

ALICE is a local-first AI smart pill dispenser designed to help people manage multiple medications safely at home.

The system combines face recognition, guided dispensing, and contextual health support to create a more private and practical home-care device for everyday medication use.

According to the project details on Devpost and my `preespp.github.io`, my contributions focused on backend API development, API research, Gemini API integration, low-level hardware integration with ESP32, and optimization for real-time performance.

My responsibilities
  • Developed backend APIs to support AI-assisted medication workflows
  • Researched and integrated Gemini API capabilities into the project stack
  • Worked on low-level hardware integration with ESP32
  • Optimized the system for more reliable real-time performance
