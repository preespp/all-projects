Engineering Portfolio

Designer & Developer

This engineering portfolio project presents experience, technical skills, and project work from across the engineering degree. It is built to serve as a clean personal hub for showcasing both classroom and outside-classroom work.

The site combines React.js, Three.js, TailwindCSS, HTML/CSS, and JavaScript to create a portfolio that is both visual and practical. It also reflects ongoing iteration through Git-based version control.

Project highlights
  • Built a dedicated space to present engineering projects, skills, and experience in one place
  • Integrated modern frontend tools including React.js, Three.js, TailwindCSS, HTML/CSS, and JavaScript
  • Used Git for iterative development and long-term maintenance of the portfolio site
