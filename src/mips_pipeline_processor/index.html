MIPS Pipeline Processor Implementation

Spring 2024

This project involved designing and implementing a 5-stage pipelined CPU and a single-cycle CPU in HDL as part of computer organization work.

The design work included hazard handling, ALU optimization, and instruction execution flow across the processor pipeline.

It was a hands-on processor architecture project connecting digital logic design with assembly-level reasoning.

My responsibilities
  • Designed and implemented a 5-stage pipelined CPU in HDL, incorporating hazard detection mechanisms
  • Optimized ALU operations within the pipeline to enhance overall processor performance and efficiency
  • Developed a single-cycle pipelined CPU in HDL, gaining proficiency in digital design and processor architecture
