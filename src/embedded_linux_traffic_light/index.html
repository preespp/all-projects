Embedded Systems Project

This project focused on building a custom Linux environment for the BeagleBone Black ARMv7 platform while also implementing and testing kernel-level interaction on real hardware.

The work included compiling the Linux kernel, configuring BusyBox, using QEMU for emulation, and developing Linux kernel modules for timers, monitoring, signaling, and GPIO-based I/O.

I also collaborated in a two-person team to implement and test a kernel-level traffic light control system on real BeagleBone hardware.

My responsibilities
  • Built a custom Linux distribution for the BeagleBone Black, including compiling the Linux kernel, configuring BusyBox, and emulating using QEMU
  • Developed multiple Linux kernel modules, including timer management with user-configurable expiration, asynchronous notifications, and a /proc interface for monitoring and signaling userspace processes
  • Programmed real-time I/O interaction in kernelspace by controlling LEDs and pushbuttons through GPIO on actual BeagleBone hardware using interrupt-driven design
  • Gained hands-on experience with cross-compilation, device drivers, kernel debugging, serial communication, and embedded Linux toolchains
  • Collaborated in a two-person team to implement and test a kernel-level traffic light control system on real hardware
