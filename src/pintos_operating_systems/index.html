Pintos Operating Systems Labs

Operating Systems, Fall 2025

Project Description

I present the Pintos lab sequence as one operating systems project because each lab builds on the same kernel codebase and progressively deepens core systems concepts. The project covered threads, user programs, system calls, virtual memory, and memory-mapped files, with file systems as an optional extension.

My Responsibilities

  • Implemented operating systems features in Pintos across scheduling, user programs, system calls, virtual memory, and memory-mapped files.
  • Worked in C on a teaching operating system codebase to strengthen understanding of kernel behavior and synchronization.
  • Built the project incrementally through a multi-lab sequence that extended the same kernel over time.

Skills I Learned

  • Kernel-level programming in C.
  • Synchronization, memory management, and process execution concepts.
  • Working systematically through layered systems assignments.
