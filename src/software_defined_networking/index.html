Software Defined Networking

Fall 2024

Project Description

This project focused on emulating a TCP/IP network using Mininet and building a software-defined networking controller with POX. The controller transformed a hub into a learning switch using OpenFlow so the network could make more intelligent forwarding decisions.

The project also involved network traffic analysis and bandwidth benchmarking, which helped connect controller logic to measurable network behavior.

My Responsibilities

  • Emulated a TCP/IP network using Mininet and developed an SDN controller in POX.
  • Implemented MAC address learning and dynamic port mapping to reduce unnecessary flooding.
  • Used tcpdump to verify packet flow and benchmark TCP bandwidth.

Skills I Learned

  • Software-defined networking concepts and OpenFlow-based control.
  • Network emulation, packet analysis, and bandwidth testing.
  • Debugging distributed behavior through traffic inspection.
