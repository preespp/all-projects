Software Defined Networking

Fall 2024

This project focused on emulating a TCP/IP network using Mininet and building a software-defined networking controller with POX.

The controller transformed a hub into a learning switch using OpenFlow, adding MAC address learning and dynamic port mapping for more efficient forwarding.

The project also involved traffic analysis and bandwidth benchmarking using tcpdump and related networking tools.

My responsibilities
  • Emulated a TCP/IP network using Mininet and developed an SDN controller in POX to convert a hub into a learning switch, enabling intelligent packet forwarding via OpenFlow
  • Implemented MAC address learning and dynamic port mapping to enhance network efficiency and reduce unnecessary packet flooding
  • Conducted network traffic analysis using tcpdump to verify packet flow and benchmarked TCP bandwidth, improving network performance visibility
