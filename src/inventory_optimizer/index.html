Inventory Optimizer

Desktop Application

Software Engineer, Technical Lead

Inventory Optimizer is an inventory management tool aimed at small retail businesses. It tracks sales and purchases to monitor inventory levels in real time and displays stock levels to the user. This system boasts a powerful backend that learns from past sales to predict future demand, then uses its predictions to compute safety stocks, ascertain reorder points, compute economic order quantities, and notify the user when a reorder is needed. Additionally, it computes and displays a variety of accounting quantities and performance metrics, including revenue, cost of goods sold, and inventory turnover ratio.

Team Members: Vugar Amirov, Zhakhangir Mamayev, Rawisara Chairat, Cole Shaigec

This project is part of Software Engineering class and focused on building a full desktop application with predictive analytics, authentication, and database-backed inventory workflows for a small business setting.

My responsibilities as technical lead
  • Developed APIs for an Inventory Management System to integrate predictive models, ensure data passage between frontend and backend, and implement hash-based authentication in Python, Node.js, and Electron.js
  • Configured MongoDB to store and retrieve inventory data and predictive data from over 40,000 data points
  • Tested and debugged the frontend and functionality of the 7-page application in HTML and JavaScript
