Room Temperature Monitor

Spring 2023

Project Description

Room Temperature Monitor is an early hardware project built to measure temperature, display the result, and alert the user when the value goes out of range. The project combined enclosure design, circuit building, soldering, and Arduino programming into one working prototype.

This project was important for me because it connected physical assembly with sensor-driven behavior and helped build my confidence in hands-on prototyping.

My Responsibilities

  • Designed interior components and circuits using Onshape and TinkerCAD.
  • Wired and soldered the circuit inside the enclosure.
  • Coded the Arduino logic to display temperature and trigger alerts for out-of-range values.

Skills I Learned

  • Basic circuit design and soldering.
  • Arduino programming for sensor-driven systems.
  • Mechanical enclosure design for small electronics projects.
