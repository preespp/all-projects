Room Temperature Monitor

Spring 2023

Room Temperature Monitor is a device built to measure room temperature with sensors, display the measured values, and alert the user with sound when needed.

The project combined enclosure design, circuit design, soldering, and Arduino programming into one complete prototype.

It was one of my earliest hands-on engineering builds that connected sensor input, hardware packaging, and user-facing output behavior.

My responsibilities
  • Designed interior components and circuits using Onshape and TinkerCAD
  • Wired and soldered the circuit inside the box enclosure
  • Coded in Arduino to operate the monitor for indicating room temperature and alerting out-of-range temperature
