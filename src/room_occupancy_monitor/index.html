Room Occupancy Monitor

Embedded Firmware Device

Computer Engineer

   Room Occupancy Monitor is a device that monitors the number of people entering/ exiting a room and prevent the trepass after reaching maximum occupancy. This is a semester long project for Engineering Design class; the team was formed by engineering students from all fields to work on different parts of project. My part is in Electrical & Computer Engineering. However, outside our specialty, we all learned general Engineering Design Principle, how to design product to suit and solve client's problem, how to apply Project Management such as Gantt Chart to our project timeline and even how to present and sell our work to client

   Team Members: Bach Nguyen, Caleb Schwartz, Camellia Mohamed

   My responsibilities
  • Programmed Firmware to control sensor data, planned C++ code flowchart, and stored code in version control (Git)
  • Optimized the components selection to reduce the client's requirement cost by 40.66%
  • Functioned IoT capabilities by supplementing an ESP32 as a WiFi interface via UART communication
  • Conducted rigorous electrical wiring, product assembly, and product testing, ensuring adherence to design specifications
  • Wrote up Preliminary Design Report and Final Reports, along with planning Gantt's chart for weekly tasks
