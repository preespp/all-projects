2-Wheel Line Follower Robot

Spring 2025

Project Description

This project involved building a two-wheel line-following robot using a Libre Computer as the embedded platform. The robot combined camera-based perception, encoder feedback, and ROS2-based software to track a path and convert sensor input into stable motion.

I enjoyed this project because it tied together controls, perception, and robotics software in one system. It also pushed me to think carefully about how to model differential-drive motion and visualize what the controller was doing in real time. And, I enjoyed working in pair with classmates in each step of the development process.

My Responsibilities

  • Developed the robot in Python and ROS2, integrating a camera for line detection and motor encoders for differential-drive modeling.
  • Implemented ROS2 nodes for image segmentation, motor commands, and PID control.
  • Published velocity commands and tracked error signals for controller tuning.
  • Built rqt and dashboard interfaces to visualize heading, linear speed, and angular speed.

Skills I Learned

  • ROS2 package design and node-based robotics software.
  • PID tuning and differential-drive kinematics.
  • Camera-based line detection and robotics debugging workflows.
  • Telemetry dashboards for real-time robot visualization.
