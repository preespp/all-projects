2-Wheel Line Follower Robot

Spring 2025

This project involved building a two-wheel line-following robot for robotics class using a Libre Computer as the embedded platform.

Development included Python and ROS2 integration, camera-based line detection, motor encoder feedback, differential-drive kinematic modeling, and PID-based control.

I also developed interfaces for live visualization of heading, linear speed, and angular speed.

My responsibilities
  • Developed a robot with Python and ROS2 on Libre Computer, integrating a camera for line detection and motor encoders for differential-drive kinematic modeling
  • Implemented a ROS2 package with nodes for image segmentation, motor commands, and a PID controller, publishing velocity commands and tracking error signals
  • Developed rqt and dashboard interfaces to visualize heading, linear speed, and angular speed
